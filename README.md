# Eye Deep-Net: Retinal Disease Diagnosis

## Overview
Eye Deep-Net is a deep learning framework for multi-class retinal disease classification using retinal fundus images.

## Publication
Published Research Paper:
https://www.researchgate.net/publication/384150211_Eye_Deep-Net_a_deep_neural_network-based_multi-class_retinal_disease_diagnostic

## Technologies Used
- Python
- TensorFlow
- Keras
- OpenCV
- NumPy
- Jupyter Notebook

## Dataset
The dataset is excluded from this repository due to GitHub storage limitations.

## Results
The model achieved high classification performance for multi-class retinal disease diagnosis.
