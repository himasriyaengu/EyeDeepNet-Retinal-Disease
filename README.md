## Project Overview

Retinal diseases are among the leading causes of vision impairment worldwide. Early and accurate diagnosis is critical for preventing irreversible vision loss.

Eye Deep-Net is a Convolutional Neural Network (CNN) based framework developed for automated multi-class retinal disease classification using retinal fundus images. The system performs image preprocessing, feature extraction, training, validation, and disease prediction in an end-to-end pipeline.

The proposed framework was designed to assist ophthalmologists and healthcare professionals by providing an intelligent decision-support system for retinal disease screening.

---

## Research Contributions

* Developed a deep learning-based retinal disease classification framework.
* Applied image preprocessing and augmentation techniques to improve model robustness.
* Implemented CNN-based feature extraction and classification.
* Evaluated performance using multiple optimization strategies.
* Conducted extensive experiments on retinal fundus image datasets.
* Achieved approximately **95% classification accuracy** for retinal disease diagnosis.

---

## Methodology

The proposed Eye Deep-Net framework consists of:

1. Dataset Loading and Visualization
2. Image Preprocessing
3. Data Normalization and Splitting
4. CNN-Based Feature Extraction
5. Model Training
6. Validation and Testing
7. Accuracy Comparison and Evaluation
8. Disease Prediction on Unseen Retinal Images

The framework utilizes deep neural network architectures optimized for retinal image analysis and multi-class disease detection.
