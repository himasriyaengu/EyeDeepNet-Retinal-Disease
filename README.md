## Abstract

Retinal diseases are among the leading causes of visual impairment and blindness worldwide. Early diagnosis is critical for preventing permanent vision loss and improving treatment outcomes. Eye Deep-Net is a deep learning-based framework designed for automated multi-class retinal disease diagnosis using retinal fundus images.

The proposed system combines image preprocessing, feature extraction, and convolutional neural network (CNN) based classification to accurately identify retinal abnormalities. The framework was developed to support ophthalmologists and healthcare professionals through intelligent computer-aided diagnosis, enabling faster and more reliable retinal disease screening.

## Methodology

The Eye Deep-Net framework follows the workflow below:

1. Dataset Acquisition and Loading
2. Retinal Image Preprocessing
3. Data Normalization and Augmentation
4. Training and Validation Split
5. CNN-Based Feature Extraction
6. Deep Learning Model Training
7. Performance Evaluation
8. Retinal Disease Prediction

The model is designed to learn discriminative retinal features directly from fundus images and classify disease categories through an end-to-end learning process.
